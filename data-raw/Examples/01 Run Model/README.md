  - A model can then be run based on the **mod** object by calling the
    function **run()** In this example, we run the model twice, and
    alter a parmeter using the **set\_params()** function

<!-- end list -->

``` r
out1 <- run(mod)
mod <- set_param(mod,"distancing_flag",1)
out2 <- run(mod)
```
